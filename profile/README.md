### 해당 organization 사용법
1. notebook(private) 레포지토리에 들어간다.
2. 상단의 Issues 탭을 클릭한다.
3. New Issue 버튼을 누른다.
4. Get started 를 클릭한다.
5. [DOCS] 다음에 제목, 하단에 내용을 입력한다.<br>
5-1. 내용은 Preview 버튼을 눌러서 미리 볼 수 있다.
6. Submit new issue 를 클릭한다.
